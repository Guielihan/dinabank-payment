# capturas de tela

nesta pasta você deve adicionar capturas de tela do projeto funcionando.

## sugestões de screenshots

- `card-preview.png` - imagem do cartão de crédito animado
- `form-validation.png` - formulário com validação em tempo real
- `card-flip.png` - animação do flip do cartão mostrando o CVV
- `mobile-view.png` - versão mobile responsiva
- `desktop-view.png` - visão completa do desktop

## como tirar as capturas

1. acesse o site: https://login-page-for-bank-f87pj6sem-guielihans-projects.vercel.app
2. tire screenshots das funcionalidades principais
3. salve os arquivos com nomes descritivos
4. adicione aqui nesta pasta

## observações

- use formato PNG ou JPG
- mantenha uma boa qualidade (mas não exagere no tamanho)
- tire screenshots em resolução adequada para visualização
- capture diferentes estados da aplicação (vazio, preenchido, com erro, etc)
